# AgenticData Quickstart

## Install

```bash
curl -fsSL https://agentralabs.tech/install/agentic-data | bash
```

## First use

```bash
# Detect format of any file
adat detect mydata.csv

# Ingest data
adat ingest mydata.csv

# Check quality
adat quality mydata

# Detect PII
adat pii mydata.csv

# List supported formats
adat formats
```
